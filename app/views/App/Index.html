Welcome to JSONStruct

JSONStruct is a small and free service designed to help you translate your JSON document into valid Golang Struct.

Paste or type your JSON document in the editor located at left side and hit Generate Go Struct bellow.


Generated JSON Struct Close Dialog

Bellow you can see JSON document output as Go Struct. In order to copy it you will need first to select all and than copy it yourself.

OS X: `cmd+c` to copy and `cmd+v` to paste.
Rest: `ctrl+c` to copy and `ctrl+v` to paste.
